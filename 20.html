Prerequisite Skills: Concepts of Communication

Critical Element Criteria:  Students must meet criteria of Critical Elements to safely perform the skill based on evidence-based-practice.  Score can be pass/fail, letter grade or point assignment.  Only acceptable score for Critical Elements is 100%

Critical Elements

  • Gather SBAR data prior to communication
  • Communicate clearly and concisely using SBAR format
  • Make appropriate recommendations given patient situation

Procedure for Nurse to Health Care Provider

Situation

  • This is _____and I’m calling about:  (patient name and location)
  • Are you familiar with this patient?
  • The problem I’m calling about is : (briefly state the problem-what is it, when it started, and how severe)

Background

  • Admitted for:
  • Pertinent History:
  • Pertinent Labs/Test Results:
  • Current Therapy: (pertinent meds, IVF, treatments, monitoring, etc.)
  • Mental Status:
  • Pertinent Assessment findings
  • Skin:
  • O2 amount/type
  • Code Status:
  • Current VS:
  • Recent I/O
  • Wt. Change

Assessment

  • This is what I think the problem is: 

OR 

  • I am not sure what the problem is, but I’m concerned:

Recommendations

  • I suggest (or request) that you:

Consider:

Are any labs or tests needed? 
Is a change in treatment needed?
Do you need additional consultations?
Is a higher level of care needed?
